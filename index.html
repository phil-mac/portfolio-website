VIRTUAL REALITY PROJECTS

PHYSICS PLAYGROUND

In Development, coming soon

The Physics Playground is being built off the framework of Curie VR, but uses a sandbox architecture for the physics simulations, so user can build and run simulations together in the classroom. It's primary use-case is for holding homeschool classes online.

CURIE VR

Completed Prototype, on SteamVR

CurieVR is is a multi-user virtual physics classroom, for use in online education. It contains interactive simulations to visualize and experiment with physics concepts, as well as learning tools such as a networked blackboard, slide deck, calculator, stopwatch, and tape measure.

ROLEPLAY U

Completed Prototype, on GitHub

RoleplayU is a free, open source app and framework for playing roleplay games like DnD in VR.

PHYSICS VERSE

Completed Prototype, not available

PhysicsVerse was an early experiment in creating a 3D networked environment to learn physics in. It explored gamifying various aspects of the education. It was built in Unity3D, used the SteamVR Interaction Template, and used the High-Level Unity networking system (UNET)
